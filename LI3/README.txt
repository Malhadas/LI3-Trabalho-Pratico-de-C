	1.Utilização da Makefile

		Estão disponíveis os seguintes comandos:
			Make GereVendas  - Compila o programa e elimina os ficheiros .o;
			Make run         - Compila, elimina os ficheiros .o e executa o programa;
			Make ValgrindRun - Executa o programa usando Valgrind com as seguintes flags ativas --leak-check=full --log-file="Valgrind.out";
			Make clean       - Elimina o executável;

	2.Utilização do programa GereVendas

              ____________________________________________________________________________
             /                                                                            \
             | Funcionalidades/Queries:                                                   |
	     |--------------------------                                                  |
	     |    |                                                     GereVendas - LI3  |
	     |----|-----------------------------------------------------------------------|
	     | 1  | Execute para inserir os ficheiros e escreva 'r' para os reinserir.    |
	     |----|-----------------------------------------------------------------------|
	     | 2  | Escreva 'q2' para obter todos produtos de uma determinada letra.      |
	     |----|-----------------------------------------------------------------------|
	     | 3  | Escreva 'q3'para obter dados das vendas e faturação de um produto     |
	     |    | globais ou de uma determinada filial sobre um mês específico.         |
	     |----|-----------------------------------------------------------------------|
	     | 4  | Escreva 'q4' para obter a lista global ou por filial dos produtos     |
	     |    | que nunca foram comprados.                                            |
	     |----|-----------------------------------------------------------------------|
	     | 5  | Escreva 'q5' para ver uma tabela com o número de produtos por mês     |
	     |    | comprados por um determinado cliente.                                 |
	     |----|-----------------------------------------------------------------------|
	     | 6  | Escreva 'q6' para obter o total de vendas e o total faturado entre    |
	     |    | um determinado intervalo de meses.                                    |
	     |----|-----------------------------------------------------------------------|
	     | 7  | Escreva 'q7' para obter todos os clientes que realizaram compras      |
             |    | em todas as filiais.                                                  |
	     |----|-----------------------------------------------------------------------|
	     | 8  | Escreva 'q8' para obter os clientes que compraram determinado         |
	     |    | produto em determinada filial.                                        |
	     |----|-----------------------------------------------------------------------|
	     | 9  | Escreva 'q9' para obter os produtos mais comprados durante um mês     |
	     |    | por um determinado cliente.                                           |
	     |----|-----------------------------------------------------------------------|
	     | 10 | Escreva 'q10'para obter a lista do N produtos mais vendidos durante   |
	     |    | o ano e dados sobre os mesmos.                                        |
	     |----|-----------------------------------------------------------------------|
	     | 11 | Escreva 'q11' para obter os N produtos em que determinado cliente     |
	     |    | gastou mais dinheiro.                                                 |
	     |----|-----------------------------------------------------------------------|
	     | 12 | Escreva 'q12' para ver o número de clientes que nunca realizaram      |
	     |    | nenhuma compra e o número de produtos que nunca foi comprado.         |
	     |----|-----------------------------------------------------------------------|
